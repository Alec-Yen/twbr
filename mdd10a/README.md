# MDD10A Motor Driver

## Overview  
This repository includes code to operate MDD10A motor driver using Pi.  
Libraries: pigpio  
Dependencies:  


## Getting Started  
Run the makefile, and execute `sudo ./bin/motor`   
To stop, type "q" and press ENTER
