INSTRUCTIONS

1. To install pigpio, run:

cd
rm pigpio.zip
sudo rm -rf PIGPIO
wget abyz.me.uk/rpi/pigpio/pigpio.zip
unzip pigpio.zip
cd PIGPIO
make
sudo make install

2. To install wiringPi, run:

cd
git clone git://git.drogon.net/wiringPi
cd wiringPi
git pull origin
./build

3. To install bcm2835, run:

cd
wget http://www.airspayce.com/mikem/bcm2835/bcm2835-1.58.tar.gz
tar zxvf bcm2835-1.58.tar.gz
cd bcm2835-1.58
./configure
make
sudo make check
sudo make install
